# Book Tracker

**Автор:** [Малинин Анатолий]

**Дата:** [26.04.2026]

## Описание

Book Tracker — это десктопное приложение для учёта прочитанных книг с графическим интерфейсом. Позволяет:
* добавлять книги (название, автор, жанр, количество страниц);
* фильтровать список по жанру и объёму (больше N страниц);
* сохранять и загружать данные в формате JSON;
* просматривать все записи в табличном виде.

## Технологии

* Python 3.6+
* Tkinter (встроенная библиотека для GUI)
* JSON (формат хранения данных)
